# 功能说明文档

## 1. 用户认证

### 登录功能
- 手机号 + 密码登录
- Token自动存储和管理
- 登录状态持久化
- 自动跳转已认证/未认证页面
- Token过期自动登出

## 2. 客户管理

### 客户列表
- 显示所有客户信息
- 支持搜索（姓名/手机号模糊查询）
- 分页显示
- 实时更新

### 客户信息字段
- 基础信息：姓名、手机号、性别、生日
- 业务信息：来源、状态、首选门店、负责员工
- 系统信息：创建时间、更新时间

### 新增客户
- 填写客户基础信息
- 选择首选门店
- 指定负责员工
- 表单验证

### 编辑客户
- 修改客户信息
- 更新首选门店
- 变更负责员工
- 修改客户状态（活跃/停用）

## 3. 履约记录管理

### 履约记录列表
- 显示所有履约记录
- 关联显示客户、门店、员工信息
- 按状态筛选（已下单/已履约/已退款）
- 金额和渠道信息展示

### 履约记录字段
- 关联信息：客户、门店、员工
- 订单信息：金额、货币、状态
- 渠道信息：支付渠道、备注
- 时间信息：支付时间、创建时间

### 新增履约记录
- 选择客户
- 选择门店
- 指定服务员工
- 填写金额
- 记录支付渠道
- 添加备注信息
- 设置支付时间

### 编辑履约记录
- 更新订单状态
- 修改备注信息
- 变更支付渠道
- 更新支付时间
- 注意：客户、门店、金额等核心信息创建后不可修改

## 4. UI/UX特性

### 响应式设计
- 自适应桌面和平板设备
- 表格横向滚动支持

### 交互反馈
- Toast消息提示
- 加载状态显示
- 错误处理和提示
- 表单验证反馈

### 视觉设计
- 现代化UI设计
- 清晰的信息层级
- 状态标签色彩区分
- 一致的操作按钮

## 5. 数据管理

### 状态管理
- React Query数据缓存
- 乐观更新
- 自动重新验证
- 错误重试机制

### API集成
- 统一的API封装
- 自动Token注入
- 错误拦截和处理
- 请求/响应转换

## 6. 字典数据

### 门店字典
- 门店列表查询
- 门店信息展示（名称、代码、城市）
- 表单下拉选择

### 员工字典
- 员工列表查询
- 按门店筛选员工
- 员工信息展示（姓名、职位）
- 表单下拉选择

## 技术亮点

1. **类型安全**: 完整的TypeScript类型定义
2. **组件化**: 可复用的UI组件库
3. **性能优化**: React Query缓存和智能更新
4. **用户体验**: 流畅的交互和及时的反馈
5. **代码质量**: 清晰的项目结构和代码组织
6. **可维护性**: 模块化的API服务和工具函数
