# 使用说明

## 登录系统

### 测试账号
在后端系统中创建测试账号后使用。

### 登录步骤
1. 打开浏览器访问 http://localhost:5173
2. 进入登录页面
3. 输入手机号和密码
4. 点击"登录"按钮
5. 登录成功后自动跳转到客户管理页面

## 客户管理

### 查看客户列表
- 登录后默认进入客户管理页面
- 显示所有客户的基本信息
- 包括：ID、姓名、手机号、性别、生日、来源、状态、创建时间

### 搜索客户
1. 在顶部搜索框中输入客户姓名或手机号
2. 系统自动进行模糊搜索
3. 实时显示搜索结果

### 新增客户
1. 点击右上角"新增客户"按钮
2. 填写客户信息：
   - **必填项**：姓名、手机号
   - **可选项**：性别、生日、来源、首选门店、负责员工、状态
3. 点击"提交"保存
4. 成功后自动刷新客户列表

### 编辑客户
1. 在客户列表中找到要编辑的客户
2. 点击该行的"编辑"按钮（铅笔图标）
3. 在弹出的对话框中修改客户信息
4. 点击"提交"保存更改
5. 成功后自动刷新客户列表

### 客户状态说明
- **活跃**：正常客户，可以正常交易
- **停用**：暂时停用的客户

## 履约记录管理

### 查看履约记录
1. 点击顶部导航栏的"履约记录"
2. 显示所有履约记录
3. 包括：ID、客户、门店、员工、金额、渠道、状态、支付时间、创建时间

### 筛选履约记录
1. 使用顶部的状态筛选器
2. 可选择：
   - 全部状态
   - 已下单
   - 已履约
   - 已退款
3. 自动更新列表

### 新增履约记录
1. 点击右上角"新增履约记录"按钮
2. 填写履约信息：
   - **必填项**：客户、门店、金额
   - **可选项**：员工、状态、渠道、备注、支付时间
3. 点击"提交"保存
4. 成功后自动刷新履约记录列表

### 编辑履约记录
1. 在履约记录列表中找到要编辑的记录
2. 点击该行的"编辑"按钮（铅笔图标）
3. 在弹出的对话框中可修改：
   - 状态
   - 渠道
   - 备注
   - 支付时间
4. 注意：客户、门店、金额等核心信息创建后不可修改
5. 点击"提交"保存更改

### 履约状态说明
- **已下单**：客户已下单，等待履约
- **已履约**：服务已完成，订单已履约
- **已退款**：订单已退款

## 系统操作

### 注销登出
1. 点击右上角的用户信息区域
2. 点击"退出"图标
3. 系统清除登录信息并跳转到登录页

### 页面导航
- 使用顶部导航栏在不同功能模块间切换
- 点击"客户管理"查看和管理客户
- 点击"履约记录"查看和管理履约记录

## 常见操作场景

### 场景1：新客户首次消费
1. 先在"客户管理"页面新增客户
2. 记录客户的基本信息和来源
3. 指定负责员工
4. 切换到"履约记录"页面
5. 为该客户创建首笔履约记录

### 场景2：老客户再次消费
1. 在"客户管理"页面搜索客户
2. 确认客户信息
3. 切换到"履约记录"页面
4. 选择该客户创建新的履约记录

### 场景3：订单退款
1. 在"履约记录"页面找到需要退款的订单
2. 点击"编辑"按钮
3. 将状态改为"已退款"
4. 在备注中填写退款原因
5. 保存更改

### 场景4：更新客户负责人
1. 在"客户管理"页面找到客户
2. 点击"编辑"按钮
3. 修改"负责员工"字段
4. 保存更改

## 数据说明

### 门店选择
- 系统会自动从后端加载门店列表
- 在创建/编辑客户和履约记录时可选择门店

### 员工选择
- 系统会自动从后端加载员工列表
- 可以为客户指定负责员工
- 可以为履约记录指定服务员工

### 金额输入
- 支持小数点后两位
- 默认货币为CNY（人民币）
- 自动进行数字格式验证

### 日期时间
- 生日使用日期选择器（年-月-日）
- 支付时间使用日期时间选择器（年-月-日 时:分）
- 创建时间由系统自动生成

## 注意事项

1. **必填字段**：标有 * 的字段为必填项，不填写无法提交
2. **手机号唯一**：客户手机号不能重复
3. **数据验证**：系统会对输入数据进行验证，请按要求填写
4. **操作反馈**：每次操作后会显示成功或失败的提示消息
5. **自动登出**：Token过期后会自动退出登录，需要重新登录
6. **网络要求**：需要确保后端API服务正常运行
